- `selectedIds` — `useState(new Set())` for checkbox selection
- `modalOpen` — `useState(false)` for publish confirmation modal

**Derived data (all useMemo):**
- `draftWorkshops` — `workshops.filter(w => w.status === 'Draft')` with dep `[workshops]`
- `conflictMap` — `buildConflictMap(workshops, coaches)` with deps `[workshops, coaches]`

- `allSelected` — `draftWorkshops.length > 0 && effectiveSelectedIds.size === draftWorkshops.length`
- `someSelected` — `effectiveSelectedIds.size > 0 && !allSelected`
- `toggleAll` — if all selected, clear set; else, set all draft IDs. Use functional setState with `new Set()`
- `toggleOne(id)` — functional setState: `new Set(prev)` then `has/delete/add`

**Select-all checkbox:** Use ref callback `ref={(el) => { if (el) el.indeterminate = someSelected; }}` to handle indeterminate state. HTML `indeterminate` is a property, not an attribute — cannot be set via JSX prop.

  <action>
Add the confirmation modal and publish handler to the DraftManager component built in Task 1.

**Publish handler (`handlePublishConfirm`):**
```
function handlePublishConfirm() {
  setWorkshops((prev) =>
    prev.map((w) =>
      effectiveSelectedIds.has(w.id) ? { ...w, status: 'Published' } : w
    )
  );
  const count = effectiveSelectedIds.size;
  setSelectedIds(new Set());
  setModalOpen(false);
  toast(`${count} workshop${count !== 1 ? 's' : ''} published`);
}
```

**Modal markup** (render at bottom of component return, after table area):

1. **Backdrop** — `{modalOpen && <div className="fixed inset-0 bg-black/30 z-20" onClick={() => setModalOpen(false)} />}`

2. **Modal container** — `{modalOpen && <div className="fixed inset-0 z-30 flex items-center justify-center p-4">}`
   - **Modal box** — `<div className="bg-white rounded-xl shadow-2xl w-full max-w-md p-6" onClick={e => e.stopPropagation()}>` (stopPropagation prevents accidental close from drag-release on backdrop)

3. **Modal content:**
   - **Title:** `<h2 className="text-lg font-semibold text-ww-navy">Publish Workshops</h2>`
   - **Description:** `<p className="mt-2 text-sm text-slate-600">You are about to publish {effectiveSelectedIds.size} workshop{s}.</p>`
   - **Conflict warning** (conditional, only when `selectedConflictCount > 0`):
