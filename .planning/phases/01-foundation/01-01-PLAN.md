<task type="auto">
  <name>Task 1: Install dependencies and configure Tailwind v4 Vite plugin</name>
  <files>package.json, vite.config.js</files>
  <action>
Run the following npm install command to add all required dependencies in one pass:

```bash
npm install tailwindcss @tailwindcss/vite react-router lucide-react date-fns @fontsource/inter
```

Then update `vite.config.js` to add the Tailwind v4 Vite plugin. The file should look like:

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})
```

IMPORTANT: Do NOT create a `tailwind.config.js` file — Tailwind v4 uses the `@theme` CSS directive instead of a JS config file. Do NOT create a `postcss.config.js` — the Vite plugin handles this.

After install, verify all six packages appear in `package.json` dependencies.
  </action>
  <verify>
Run `npm ls tailwindcss @tailwindcss/vite react-router lucide-react date-fns @fontsource/inter` — all six should be listed without errors.
Run `cat vite.config.js` — should import and register both `react()` and `tailwindcss()` plugins.
  </verify>
  <done>All six packages installed in package.json dependencies; vite.config.js registers both React and Tailwind v4 plugins.</done>
