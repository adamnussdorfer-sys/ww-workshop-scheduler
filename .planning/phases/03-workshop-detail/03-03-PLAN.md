  - Add `className` addition: `cursor-pointer` (append to existing classes)
  - Add `onClick` handler with 30-minute snap geometry:
    ```
    onClick={(e) => {
      // Only fire when clicking the background, not a workshop card
      if (e.target !== e.currentTarget) return;
      const rect = e.currentTarget.getBoundingClientRect();
      // Account for scroll offset of the parent overflow container
      const scrollContainer = e.currentTarget.closest('.overflow-y-auto');
      const scrollTop = scrollContainer ? scrollContainer.scrollTop : 0;
      const relY = e.clientY - rect.top + scrollTop;
      const totalMinutes = Math.floor(relY / PX_PER_MIN);
      const snappedMinutes = Math.floor(totalMinutes / 30) * 30;
      const hour = GRID_START_HOUR + Math.floor(snappedMinutes / 60);
      const minute = snappedMinutes % 60;
      // Clamp to grid bounds
      if (hour >= GRID_START_HOUR && hour < GRID_END_HOUR) {
        onSlotClick?.(day, hour, minute);
      }
    }}
    ```
  - The `e.target !== e.currentTarget` guard prevents firing when clicking a workshop card (WorkshopCard already calls stopPropagation from Plan 01, but this is defense-in-depth)
  - Note: `scrollContainer.scrollTop` handles the offset when the grid body is scrolled (the `overflow-y-auto` div wrapping day columns)

Create `src/utils/coachAvailability.js`:
- Export function `getCoachAvailability(coach, date, startHour, startMinute)`:
  - `date` is a JS Date object for the workshop day
  - Import `format` from date-fns
  - Compute day name: `format(date, 'EEEE').toLowerCase()` — returns 'monday', 'tuesday', etc., matching the data format exactly (lowercase)
  - Find matching availability slot: `coach.availability.find(a => a.day === dayName)`
  - If no slot: return `{ available: false, reason: 'No availability on ' + format(date, 'EEEE') }`
  - Parse slot times: split `slot.start` and `slot.end` on ':' to get hours and minutes, convert to total minutes
  - Compute workshop start in total minutes: `startHour * 60 + startMinute`
  - If workshop start < availability start: return `{ available: false, reason: 'Available from ' + slot.start }`
  - If workshop start >= availability end: return `{ available: false, reason: 'Available until ' + slot.end }`
  - If `coach.status === 'inactive'`: return `{ available: false, reason: 'Inactive coach' }`
  - Otherwise: return `{ available: true, reason: null }`
