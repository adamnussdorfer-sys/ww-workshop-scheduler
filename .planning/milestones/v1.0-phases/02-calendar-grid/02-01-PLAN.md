**CalendarGrid.jsx:**
- Receives `weekDays` (array of 7 Date objects, Mon-Sun), `workshops` (array), `coaches` (array) as props.
- Build `coachMap` via `useMemo(() => new Map(coaches.map(c => [c.id, c])), [coaches])`.
- Filter out cancelled workshops: `workshops.filter(ws => ws.status !== 'Cancelled')`.
- Constants: `GRID_START_HOUR = 6`, `GRID_END_HOUR = 22`, `PX_PER_HOUR = 64`, `PX_PER_MIN = PX_PER_HOUR / 60`, `GRID_HEIGHT = (22 - 6) * 64` (1024px).
- `getEventPosition(startTimeISO, endTimeISO)` function: compute `top` from `(getHours(start) - GRID_START_HOUR) * 60 + getMinutes(start)` times PX_PER_MIN, compute `height` from `differenceInMinutes(end, start) * PX_PER_MIN` with min 20px and clamped to not exceed GRID_HEIGHT.
- Layout: flex container with time gutter (w-16, flex-shrink-0) on left, then 7 day columns in a CSS grid (`grid-template-columns: repeat(7, 1fr)`).
- **Day header row** (sticky at top of each column): display day name (`format(day, 'EEE')`) and date number (`format(day, 'd')`). Height 40px, border-b. Use `isToday` from date-fns to highlight today's column header with `bg-ww-blue/10 text-ww-blue font-bold`.
- **Time gutter**: 40px spacer at top (aligned with day header), then relative container of GRID_HEIGHT. Hour labels from 6 to 22 positioned absolutely at `(h - GRID_START_HOUR) * PX_PER_HOUR` offset. Format: `h === 12 ? '12 PM' : h < 12 ? h + ' AM' : (h - 12) + ' PM'`.
- **Day columns event area**: relative container of GRID_HEIGHT. Render 32 horizontal slot row lines (every 32px for 30-min slots) — hour lines use `border-border`, half-hour lines use `border-border/50`. For each day, filter workshops by `isSameDay(parseISO(ws.startTime), day)` and render each as an absolutely positioned div containing WorkshopCard. Position: `top` from getEventPosition, `height` from getEventPosition, `left: 2px`, `right: 2px`. For overlapping events on the same day, add a simple horizontal offset: each card at index `idx` gets `left: 2 + idx * 4` pixels (makes stacked cards slightly visible).
- Import date-fns: `parseISO, getHours, getMinutes, differenceInMinutes, isSameDay, format, isToday`.
- Outer container: `border border-border rounded-lg overflow-hidden bg-white`.
